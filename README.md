## ğŸƒâ€â™‚ï¸ Getting Started

Follow these steps to run the project locally.

### Prerequisites

Ensure you have **Node.js** assigned (v18+ recommended) and `npm` installed.

### Installation

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/nipunlnick/nipun-lakshitha-portfolio.git
    cd nipun-lakshitha-portfolio
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run the development server**:

    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to view the app.
